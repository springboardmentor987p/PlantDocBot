##📚 Colab Notebooks

This directory contains the original Google Colaboratory notebooks used to perform data preprocessing, model training, and asset generation for the PlantDoc Bot application.

These notebooks serve as the complete documentation for the machine learning pipeline and asset creation.

##⚙️ Key Technical Details

-Model Validation: Each notebook includes metrics (accuracy, loss) to validate the model's performance on the test set.

-Preprocessing Steps: Details are provided on image transformations (resizing to 128x128, normalization) and text tokenization (max length 256).

-Asset Saving: Documents the exact steps used to save the final deployment assets (.pth and Hugging Face directories) required by the FastAPI server.

⚠️ Note on Deployment
The final prediction assets generated by these notebooks are placed directly into the project root (plant_cnn.pth) and the text_model_artifacts/ folder for use by main.py. These notebooks themselves are not run during API deployment.
