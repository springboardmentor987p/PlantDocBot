## 📥 Model Setup

The machine learning models for this project are too large for GitHub and must be downloaded separately.

1.  **Download the models** from this link:
    For image classification model download this file like cnn.model.pth (https://drive.google.com/file/d/1MXMz-hBrqbqKnczM486nkbkY2rbSzaiU/view?usp=sharing)
    for text model, download these files like best_model_text (https://drive.google.com/drive/folders/1aqKHcEMsyzvO4ym0kHpsig3AoS1XOZ9L?usp=sharing) & (https://drive.google.com/file/d/1jZA8RYyh3S6iIzAWpEm90Rsk7mtgxhzP/view?usp=sharing)

2.  **download the file** to get the `Models` folder.

3.  **Place the `Models` folder** inside the `backend/` directory of this project.

✔️ Your final folder structure should look like this:
backend/
├── Models/ <-- The folder you just downloaded and placed
│ ├── cnn_model.pth
│ └── best_model_text/
└── main.py
