# PlantDocBot: AI Plant Disease Diagnosis System

PlantDocBot is an AI-powered application that helps users quickly and accurately diagnose plant diseases. It supports two input methods: uploading images of diseased leaves or describing symptoms in text. The system provides predicted disease classes along with confidence scores and expert-verified treatment recommendations.

The project features a full-stack architecture with a **FastAPI backend** serving a **CNN-based image prediction model** and a **BERT-based text classification model**, and a **React.js frontend** providing a modern, responsive user interface. Large model files are hosted on Google Drive due to GitHub size limits.

PlantDocBot enables farmers, gardeners, and agricultural workers to make informed decisions and take timely action to prevent crop loss, combining **machine learning** and **web technologies** into a real-world solution.
